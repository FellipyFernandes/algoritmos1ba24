Iniciando tutorial de Python:

-O Python é uma linguagem de programação de alto nível e que tem como principal princípio permitir
uma alta legibilidade de código através de uma sintaxe simples, porém, poderosa e que permite que
programadores desenvolvam suas soluções utilizando poucas linhas de código. A linguagem Python foi
desenvolvida por Guido Van Rossum no final dos anos 80.

Explicando o que é Python:

-Python é uma linguagem de programação de alto nível e de uso geral. 
Sua filosofia de design enfatiza a legibilidade do código com o uso de recuo significativo. 
Python é digitado dinamicamente e coletado como lixo. Ele suporta vários paradigmas de programação,
incluindo programação estruturada, orientada a objetos e funcional.

Como configurar o virtualenv:

Virtualenv é uma ferramenta que permite criar ambientes Python isolados. Isto é útil para gerenciar dependências e evitar conflitos entre diferentes projetos.

1. Abra um terminal ou prompt de comando e navegue até o diretório do seu projeto.
2. Digite o seguinte comando para instalar o virtualenv:
	1. pip install virtualenv
3. Depois que o virtualenv estiver instalado, digite o seguinte comando para criar um novo ambiente virtual:
	1. virtualenv venv
4. Para ativar o ambiente virtual, digite o seguinte comando:
	1. source venv/bin/activate (macOS/Linux)
	2. venv\Scripts\activate (Windows)
5. Depois que o ambiente virtual for ativado, você poderá instalar pacotes Python usando pip sem afetar a instalação global do Python.
